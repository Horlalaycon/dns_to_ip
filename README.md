# domain_name_to_ip Converter
A program to convert Domain name to ip address

## usage
### python3 dns_to_ip -d example.com

## Installation
- ```git clone https://github.com/Horlalaycon/domain_to_ip.git```
- ```cd domain_to-ip```
- ```python3 dns_to_ip -d example.com```
